# Concept

> Detailed concept notes for PAN LAN — DIGITAL ARCHIVE.
> Authoritative rules live in [PROJECT_SPEC.md](../PROJECT_SPEC.md).

## What This Is

A personal research archive — a structured, evolving index of work across design, research, writing, teaching, and experimentation.

It is **not** a portfolio website. There is no pressure to impress on first visit. The site rewards return visits and deep reading.

## Archive Domains

Future content categories (added incrementally):

| Domain              | Description                                  |
| ------------------- | -------------------------------------------- |
| Research            | Academic and independent research projects   |
| Publications        | Papers, articles, and published work         |
| Installations       | Spatial and exhibition work                  |
| Writing             | Essays, notes, and long-form text            |
| Teaching            | Courses, workshops, and pedagogical material |
| Experiments         | Exploratory and process-driven work          |
| Future PhD Research | Doctoral research (forthcoming)              |

## Landing Page

The entry point presents identity through typography alone:

```
PAN LAN
DIGITAL ARCHIVE
2022–PRESENT
```

No imagery, no navigation clutter. The landing page establishes tone and signals that this is an ongoing record.
